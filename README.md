## Guia de instalação

### Pré-requisitos

- Docker
- Docker Compose

---

### 1. Clonar o repositório

```bash
git clone https://github.com/annaiwiny/sistema-aviario.git
cd sistema-aviario
```

### 2. Executando a aplicação
    
```bash
docker compose up --build
```

### 3. Acesse a aplicação no navegador
    
```bash
http://localhost:3000
```
